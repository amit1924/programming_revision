All Properties

Complete reference of Flexbox properties for the container and items, including the flex shorthand.

Flex Container Properties

Property Values Description
displayflex | inline-flexCreates a flex container
flex-directionrow (default) | row-reverse | column | column-reverseSets main axis direction
flex-wrapnowrap (default) | wrap | wrap-reverseAllow items to wrap onto multiple lines
flex-flow<flex-direction> <flex-wrap>Shorthand for direction + wrap
justify-contentflex-start | flex-end | center | space-between | space-around | space-evenlyMain axis alignment
align-itemsstretch (default) | flex-start | flex-end | center | baselineCross axis alignment (single line)
align-contentstretch (default) | flex-start | flex-end | center | space-between | space-aroundCross axis alignment (multi-line, needs wrap)

Flex Item Properties

Property Values Description
orderinteger (default 0)Visual order of the item
flex-grownumber (default 0)Ability to grow if space available
flex-shrinknumber (default 1)Ability to shrink if not enough space
flex-basislength | auto (default)Initial size before grow/shrink
flexgrow shrink basis | auto | noneShorthand for grow, shrink, basis
align-selfauto (default) | flex-start | flex-end | center | baseline | stretchOverride align-items for this item

Flex Shorthand Reference

flex: initial /* 0 1 auto — default */
flex: auto /* 1 1 auto — grows and shrinks */
flex: none /* 0 0 auto — fully inflexible */
flex: <number> /* <number> 1 0 — e.g. flex: 2 */
flex: 1 2 10% /* grow: 1, shrink: 2, basis: 10% */
