Flex Direction

flex-direction sets the main axis direction: row (default, left→right), row-reverse (right→left), column (top→bottom), column-reverse (bottom→top).

Direction Selector

flex-direction: row
→ Main axis (row) →
1
2
3
4

Property Values

row (default)
Left to right. Main axis is horizontal.
row-reverse
Right to left. Main axis is horizontal, reversed.
column
Top to bottom. Main axis is vertical.
column-reverse
Bottom to top. Main axis is vertical, reversed.
