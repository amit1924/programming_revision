Flex Container

A flex container is created with display: flex or display: inline-flex. All direct children become flex items. The container controls the layout along the main axis and cross axis.

Main Axis & Cross Axis

The main axis depends on flex-direction. The cross axis is perpendicular.

← Main Axis (row) →
↑ Cross Axis ↓
← Cross Axis →
↓ Main Axis (column) ↓

Basic Flexbox Setup

.container {
display: flex; /* or inline-flex */
flex-direction: row; /* column | row-reverse | column-reverse */
justify-content: flex-start; /* main axis alignment */
align-items: stretch; /* cross axis alignment */
}
