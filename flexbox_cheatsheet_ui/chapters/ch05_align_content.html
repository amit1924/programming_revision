Align Content

align-content distributes space between multiple lines of flex items along the cross axis. Only works when flex-wrap: wrap is set and items span multiple lines.

All Values

stretch (default)
Lines stretch to take up remaining space
flex-start
Lines packed to the start of the cross axis
flex-end
Lines packed to the end of the cross axis
center
Lines centered along the cross axis
space-between
First line at start, last at end, equal space between
space-around
Equal space around each line
