Flex Wrap & Flow

flex-wrap controls whether items stay on one line (nowrap) or wrap onto multiple lines (wrap, wrap-reverse). flex-flow is shorthand for flex-direction + flex-wrap.

Values

nowrap (default)
All items stay on one line (may overflow)
wrap
Items wrap to new lines below if needed
wrap-reverse
Items wrap to new lines above if needed
flex-flow
Shorthand: flex-direction flex-wrap
