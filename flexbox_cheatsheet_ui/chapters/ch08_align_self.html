Align Self

align-self overrides the container's align-items for a single flex item. The container below has align-items: stretch — the middle item overrides it.

Interactive Demo

Container has align-items: stretch (all items stretch). The middle item overrides with align-self:

Item 1
(default)
Item 2
align-self: auto
Item 3
(default)
align-self on Item 2: auto

Values

auto (default)
Inherits the parent's align-items value
flex-start
Cross-start margin is on the cross start line
flex-end
Cross-end margin is on the cross end line
center
Item centered along the cross axis
baseline
Item's baseline aligned with others
stretch
Stretch to fill container height
