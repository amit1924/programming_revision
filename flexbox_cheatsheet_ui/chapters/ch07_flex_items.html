Flex Items

Individual flex items can be customized with order, flex-grow, flex-shrink, and flex-basis. These override the container-level behavior for a specific item.

Order

Change the visual order of items. Default is 0. Lower numbers come first.

Item 1
Item 2
Item 3

Flex Grow

Items with flex-grow > 0 will take up extra space proportionally.

grow: 0
grow: 0
grow: 0

Flex Shrink & Basis

flex-shrink controls how an item shrinks when there isn't enough space. flex-basis sets the initial size before growing/shrinking.

flex: 1 1 100px
flex: 1 1 150px
flex: 1 1 100px

Flex Shorthand

flex: flex-grow flex-shrink flex-basis;
/* example: flex: 2 2 10%; */

/* Common shorthands: */
flex: 1; /* flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
flex: auto; /* 1 1 auto */
flex: none; /* 0 0 auto */
