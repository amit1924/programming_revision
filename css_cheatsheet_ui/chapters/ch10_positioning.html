Positioning

CSS positioning controls how elements are placed in the document. position values: static, relative, absolute, fixed, sticky. Use top, right, bottom, left, z-index to place elements.

Positioning Properties

position static | relative | absolute | fixed | sticky
top / right / bottom / left auto | length | %
z-index auto | number (stacking order)
clip auto | shape() (clips visible area)
float left | right | none
clear left | right | both | none
