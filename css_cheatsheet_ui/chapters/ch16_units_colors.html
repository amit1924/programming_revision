Units Reference

px Pixel (1/96th of 1in)
% Percentage of parent
em Relative to parent font-size
rem Relative to root (html) font-size
vw 1% of viewport width
vh 1% of viewport height
vmin / vmax Smaller/larger of vw and vh
ch Width of "0" character in current font
ex x-height of current font
cm / mm / in Physical units
pt / pc Point (1/72in) / Pica (12pt)
deg / rad / grad / turn Angle units
s / ms Time units
Hz / kHz Frequency units

Color Formats Reference

#rrggbb Hex color (e.g. #ff0000 = red)
#rgb Shorthand hex (e.g. #f00 = red)
rgb(r,g,b) 0-255 each channel
rgba(r,g,b,a) With alpha 0-1
hsl(h,s%,l%) Hue 0-360, Sat/Light %
hsla(h,s%,l%,a) With alpha
named colors red, blue, green, purple, etc.
currentColor Inherits current color value
transparent Fully transparent
accent-color User's system accent color
