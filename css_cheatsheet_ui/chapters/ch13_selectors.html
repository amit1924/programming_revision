Selectors

CSS selectors target HTML elements for styling: type, class, ID, attribute, combinators, and the universal selector.

Universal, Type, Class & ID

* /* Universal — targets everything */

h2 /* Type selector */ → This is an h2

.highlight /* Class selector */ → highlighted text

#unique /* ID selector */ → Only one per page

Attribute Selectors

[data-type] Any element with data-type attribute:

data-type="info" — purple border
data-type="primary" — blue border
data-type="warning" — amber border

[attr] [attr=val] [attr~=val] [attr^=val] [attr$=val] [attr*=val]

Selector Reference

* Universal — any element
h1 Type — any element of that type
.class Class — multiple elements sharing a class
#id ID — a single unique element
div p Descendant — p inside div (any depth)
div > p Child — direct child p of div
h1 + p Adjacent sibling — p immediately after h1
h1 ~ p General sibling — any p after h1
[attr] Has attribute
[attr=val] Exact attribute value
[attr^=val] Attribute starts with val
[attr$=val] Attribute ends with val
[attr*=val] Attribute contains val
h1, h2, h3 Grouping — multiple selectors
