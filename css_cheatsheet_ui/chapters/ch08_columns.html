CSS Multi-Column Layout

CSS multi-column layout allows you to arrange content into columns, similar to a newspaper. This is useful for long text passages, lists, and galleries.

The column-count property sets the number of columns. column-gap controls space between columns. column-rule adds a visible dividing line.

You can also use column-width to set a minimum width and let the browser auto-calculate the count. column-span makes an element span all columns.
