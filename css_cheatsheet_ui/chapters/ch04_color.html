Color

CSS supports multiple color formats: named colors, hex, rgb, rgba, hsl, and hsla. Opacity controls transparency.

color Sets text color (named | hex | rgb | hsl | etc.)
opacity 0.0 (transparent) to 1.0 (opaque)
Hex #rrggbb or #rgb (e.g. #ff0000 or #f00)
RGB rgb(red, green, blue) — 0-255 each
RGBA rgba(r, g, b, alpha) — alpha 0-1
HSL hsl(hue, saturation%, lightness%)
HSLA hsla(hue, sat%, light%, alpha)
Named Colors red, blue, green, purple, etc.
