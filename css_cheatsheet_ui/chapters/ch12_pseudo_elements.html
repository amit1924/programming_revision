::first-letter & ::first-line

CSS pseudo-elements let you style specific parts of an element. The first letter is styled with a larger pink font using ::first-letter. The first line is bold and gold using ::first-line. Try selecting text to see ::selection in action — it changes selected text to white on purple.

Pseudo-element Reference

::before Inserts content before an element
::after Inserts content after an element
::first-letter Styles the first letter of text
::first-line Styles the first line of text
::selection Styles selected/highlighted text
::placeholder Styles input placeholder text (modern)
::marker Styles list markers (modern)
