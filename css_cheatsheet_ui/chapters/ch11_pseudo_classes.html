Pseudo-class Reference

:hover Mouse over element
:focus Element has focus
:active Element being activated
:link Unvisited link
:visited Visited link
:disabled / :enabled Form element state
:checked Checked checkbox/radio
:nth-child(n) Nth sibling
:nth-last-child(n) Nth from last sibling
:first-child / :last-child First/last sibling
:nth-of-type(n) Nth sibling of its type
:first-of-type / :last-of-type First/last of type
:only-child / :only-of-type Only child
:empty Element with no children
:root Root element (html)
:not(x) Not matching selector x
:target Targeted by URL fragment
:lang(language) Element with given language
