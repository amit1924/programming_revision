Z-Index & Order

Grid items can occupy the same cell. Use z-index to control stacking order. The order property changes visual order without affecting DOM order.

Z-Index — Overlapping Items

Three items occupy the same cell. Adjust z-index to change which appears on top.

Red z: 1
Blue z: 2
Green z: 3

Items later in source order appear on top by default. Higher z-index = on top.

Order Property

Change the visual order of grid items without changing HTML.

Items are re-ordered visually by the order property. The DOM stays unchanged.

Properties

z-index Controls stacking order (higher = on top)
order Changes visual order within the grid
Items can occupy same cells when positioned via lines/areas
Items lower in source order appear below unless z-index is used
/* Overlapping items with z-index */
.box1 { grid-column: 1; grid-row: 1; z-index: 1; }
.box2 { grid-column: 1; grid-row: 1; z-index: 2; }
/* box2 appears on top of box1 */
