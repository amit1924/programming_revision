Grid Container

CSS Grid Layout introduces a two-dimensional grid system. Define a grid container with display: grid or display: inline-grid. All direct children become grid items.

Terminology

Grid Container

The parent element with display: grid or display: inline-grid. All direct children become grid items.

Grid Cell

The smallest unit on a grid. Conceptually like a table cell. Child items lay themselves out in one cell each of the defined grid.

Grid Tracks

Defined by grid-template-columns and grid-template-rows. A grid track is the space between any two lines on the grid. There are column tracks and row tracks.

Grid Lines

Numbered lines used for positioning items. In a 5-column, 3-row grid there are 6 column lines and 4 row lines (number of tracks + 1).

Grid Area

Items can span one or more cells both by row or column, creating a grid area. Grid areas must be rectangular.

Gutters (grid-gap)

Alleys between grid cells created with grid-gap, grid-column-gap, and grid-row-gap properties.

Basic Grid Code

.container {
display: grid;
grid-template-columns: repeat(3, 1fr);
gap: 10px;
}

.item {
background: #3b82f6;
padding: 1rem;
}
