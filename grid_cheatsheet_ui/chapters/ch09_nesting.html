Nesting Grids

A grid item can become a grid container itself. The nested grid has no relationship to the parent. Gutters and lines in the nested grid do not align with the parent.

Key Points

✦ Any grid item can become a grid container by setting display: grid on it

✦ The nested grid is independent of the parent grid

✦ Grid lines of parent and nested grid do not align

✦ gap of the parent and nested grid are separate

✦ Useful for complex layouts like cards with internal structure

/* Parent grid */
.parent {
display: grid;
grid-template-columns: repeat(3, 1fr);
}

/* Nested grid inside a grid item */
.item {
display: grid;
grid-template-columns: 1fr 1fr;
gap: 10px;
}
