Grid Container Properties

Property Values Description
displaygrid | inline-gridCreates a grid container
grid-template-columnstrack-size ... | repeat()Defines column tracks
grid-template-rowstrack-size ... | repeat()Defines row tracks
grid-template-areasstring names | noneNamed grid areas
grid-templaterows / cols | areasShorthand for template
column-gaplength | normalGap between columns
row-gaplength | normalGap between rows
gaprow-gap column-gapShorthand for gaps
grid-auto-columnstrack-sizeSize of implicit columns
grid-auto-rowstrack-sizeSize of implicit rows
grid-auto-flowrow | column | denseAuto-placement algorithm
gridshorthandMega shorthand for all grid props
justify-itemsstart | end | center | stretchAlign items horizontally
align-itemsstart | end | center | stretchAlign items vertically
place-itemsalign justifyShorthand for both align/justify items
justify-contentstart | end | center | stretch | space-around | space-between | space-evenlyAlign grid horizontally
align-contentstart | end | center | stretch | space-around | space-between | space-evenlyAlign grid vertically
place-contentalign justifyShorthand for content alignment

Grid Item Properties

Property Values Description
grid-column-startline-number | span NStart column line
grid-column-endline-number | span NEnd column line
grid-row-startline-number | span NStart row line
grid-row-endline-number | span NEnd row line
grid-columnstart / endShorthand
grid-rowstart / endShorthand

align-selfstart | end | center | stretchAlign item vertically in cell
place-selfalign justifyShorthand for item alignment
orderintegerVisual order (default 0)

Functions & Keywords

repeat(count, pattern) Repeat track patterns
minmax(min, max) Track size range
fit-content(limit) Clamp to limit, but auto-shrink
span N Span across N tracks
fr Fraction of available space
auto-fill Fill with as many tracks as possible
auto-fit Fit tracks, collapsing empty ones
min-content Smallest size without overflow
max-content Size needed for all content

Legacy Properties (Still Supported)

grid-column-gap → use column-gap
grid-row-gap → use row-gap
grid-gap → use gap
