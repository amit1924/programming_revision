Positioning Items

Place items precisely on the grid using grid-column, grid-row, and line numbers. Span multiple tracks with span keyword.

grid-column-start Start line number
grid-column-end End line number
grid-row-start Start row line
grid-row-end End row line
grid-column Shorthand: start / end
grid-row Shorthand: start / end
span Keyword: span N tracks

Positioning Examples

/* Place between lines 1 and 3 */
grid-column: 1 / 3;

/* Start at line 2, span 3 tracks */
grid-column: 2 / span 3;

/* Place in row 1, columns 1-4 */
grid-row: 1;
grid-column: 1 / 5;
