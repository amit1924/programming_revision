Grid Areas

grid-template-areas names regions of the grid. Place items into named areas with grid-area. This makes complex layouts intuitive and readable. Grid areas must be rectangular.

Grid Area CSS

.container {
display: grid;
grid-template-areas:
'header header header'
'sidebar main aside'
'footer footer footer';
grid-template-columns: 200px 1fr 200px;
grid-template-rows: auto 1fr auto;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

Grid Area Properties

grid-template-areas Define named grid areas with string syntax
grid-area Assign item to a named area, or provide row-start / col-start / row-end / col-end
. (dot) Empty cell in grid-template-areas
grid-template Shorthand for rows / columns / areas
/* Grid area with dots for empty cells */
grid-template-areas:
'header header header'
'sidebar . aside'
'footer footer footer'
