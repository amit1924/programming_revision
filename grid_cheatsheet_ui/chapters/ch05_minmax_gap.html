minmax() & Gap

minmax(min, max) gives tracks a minimum and maximum size. gap (formerly grid-gap) creates gutters between grid cells.

Properties

minmax(min, max) min size / max size for tracks
gap Shorthand for row-gap and column-gap
row-gap Gap between rows (formerly grid-row-gap)
column-gap Gap between columns (formerly grid-column-gap)
grid-gap Legacy shorthand for gap (still supported)
