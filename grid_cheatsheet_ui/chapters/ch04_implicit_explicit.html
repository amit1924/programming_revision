Implicit & Explicit Grid

The explicit grid is defined with grid-template-columns and grid-template-rows. The implicit grid is auto-generated when content exceeds the explicit grid. Control implicit tracks with grid-auto-rows and grid-auto-columns.

How It Works

/* Explicit: define 3 columns */
grid-template-columns: repeat(3, 1fr);

/* Implicit: auto-sized rows for extra content */
grid-auto-rows: 100px; /* fixed size */
grid-auto-rows: minmax(100px, auto); /* flexible */

/* Auto columns for extra columns */
grid-auto-columns: 150px;

Key Points

✦ Explicit grid — you define column/row count with grid-template-columns and grid-template-rows

✦ Implicit grid — auto-created rows/columns when there's more content than the explicit grid

✦ grid-auto-rows — set size for implicitly created rows

✦ grid-auto-columns — set size for implicitly created columns

✦ Without grid-auto-rows, implicit tracks are auto-sized based on content

✦ minmax(min, max) works well with implicit tracks for flexible sizing
