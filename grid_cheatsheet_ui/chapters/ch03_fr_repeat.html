fr Unit & repeat()

The fr unit represents a fraction of the available space in the grid container. It grows and shrinks proportionally. repeat() lets you define repeating track patterns concisely.

fr Unit — Proportional Sizing

grid-template-columns: 1fr 1fr 1fr
1fr
1fr
1fr

fr units divide available space proportionally. Combine with fixed px for hybrid layouts.

repeat() Notation

grid-template-columns: repeat(3, 1fr)
col 1
col 2
col 3
col 4
col 5
col 6
/* 3 columns each with 1 fraction of available space */
grid-template-columns: repeat(3, 1fr);

/* 8 cols: first 20px, 6×1fr, last 20px */
grid-template-columns: 20px repeat(6, 1fr) 20px;

/* 10 cols: 5× (1fr + 2fr) pattern */
grid-template-columns: repeat(5, 1fr 2fr);

Key Points

✦ fr = fraction of available space (after fixed-size tracks)

✦ Use fr for responsive layouts that adapt to any viewport

✦ repeat(count, pattern) repeats track definitions

✦ Combine fixed and flexible: 200px repeat(2, 1fr) 200px

✦ Resize your browser to see how fr units fluidly adjust
