Grid Tracks

Grid tracks are defined with grid-template-columns and grid-template-rows. A track is the space between any two grid lines. You can use fixed units (px), flexible units (fr, %), or the minmax() function.

Key Concepts

✦ Fixed tracks use px, pt, cm — grid stays at exact sizes regardless of viewport.

✦ Flexible tracks use %, fr — grid adapts to available space.

✦ The fr unit represents a fraction of available space in the grid container.

✦ fr grows and shrinks proportionally. Combine with fixed sizes freely.

✦ repeat() notation repeats track patterns. E.g. repeat(3, 1fr) = 1fr 1fr 1fr.

✦ Grid lines are automatically numbered — track count + 1 lines per axis.
