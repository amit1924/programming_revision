# 📚 Programming Revision

Interactive visual cheatsheets for HTML, CSS, Tailwind, Flexbox, Grid, React, and Python — all in one place.

## 🔗 Live Site

👉 **[View the Master Index](https://amit1924.github.io/programming_revision/)**

## 📖 What's Inside

| Cheatsheet | Description | Tech |
|---|---|---|
| [HTML](html_cheatsheet_ui/) | All elements, tags, attributes, entities — 154 interactive cards | Vanilla JS |
| [CSS](css_cheatsheet_ui/) | Selectors, properties, box model, animations | Vanilla JS |
| [Tailwind CSS](tailwind_cheatsheet_ui/) | 15 chapters covering all utility classes | Tailwind CDN |
| [Flexbox](flexbox_cheatsheet_ui/) | Container & item properties, alignment, wrapping | Vanilla JS |
| [Grid](grid_cheatsheet_ui/) | Template, areas, gaps, auto-flow playground | Vanilla JS |
| [React](react_cheatsheet_ui/) | JSX, Components, Hooks, Class lifecycle, Error Boundaries | React + Vite |
| [Python](python_cheatsheet_ui/) | Basics, data structures, functions, OOP | Vanilla JS |

## 🚀 Local Development

```bash
# Open the master index in your browser
open index.html

# For the React cheatsheet
cd react_cheatsheet_ui
npm install
npm run dev
```
