Git

Setup

$ git --version
$ git config --global user.name "eric"
$ git config --global user.email "eric@example.com"
$ git config --global core.editor "nano"
git version 2.30.1

Basic Workflow

$ git init
$ git status
$ git add .
$ git commit -am "Started project, everything works."
2 files changed, 7 insertions(+)
          create mode 100644 .gitignore
          create mode 100644 hello.py

History & Navigation

$ git log --oneline
dc2ebd6 Greets user.
          bf55851 Started project.
$ git checkout b9aedbb # visit specific commit
$ git checkout main # back to latest
$ git checkout HEAD~2 # 2 commits back

Branching

$ git checkout -b new_feature # create & switch
$ git branch # list branches
$ git checkout main # switch back
$ git merge new_feature # merge to main
$ git branch -D new_feature # delete branch
Switched to a new branch 'new_feature'
          * main
            new_feature
          Updating b9aedbb..5e5130a (Fast-forward)

Diff (See Changes)

$ git diff # all changes
$ git diff hello.py # one file
$ git diff HEAD~2 # since 2 commits ago
$ git diff fab2cdd 7c0a5d8 # between two commits
diff --git a/hello.py b/hello.py
          + print("Hello!")

Stashing

$ git stash # save changes for later
$ git stash list
$ git stash pop # reapply latest
$ git stash clear # remove all
Saved working directory and index state WIP on main: f6f39a6...

Undoing Changes

$ git checkout . # discard uncommitted
$ git reset --hard b9aedbb # reset to that commit
$ git reset --hard HEAD~1 # undo last commit

GitHub

$ git clone https://github.com/user/repo.git

$ git remote add origin https://github.com/user/repo.git
$ git push -u origin main
Enumerating objects: 10, done.
          * [new branch] main -> main
$ git push origin my_branch
