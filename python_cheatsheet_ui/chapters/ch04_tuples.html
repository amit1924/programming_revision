Tuples

A tuple is like a list, but immutable — you cannot change values once defined. Use parentheses ().

Defining a Tuple

dimensions = (800, 600)
print(dimensions)
(800, 600)

Looping Through

dimensions = (800, 600)
for dim in dimensions:
    print(dim)
800
          600

Overwriting (not modifying!)

dimensions = (800, 600)
print(dimensions)
dimensions = (1200, 900) # new tuple
print(dimensions)
(800, 600)
          (1200, 900)
dimensions[0] = 1200 would ERROR — tuples are immutable!
