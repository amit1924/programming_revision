Variables & Strings

Variables assign labels to values. A string is a series of characters surrounded by quotes. Python's f-strings let you embed variables in strings.

Hello World

print("Hello world!")
Hello world!

Variables

msg = "Hello world!"
print(msg)
Hello world!

f-strings

first_name = 'albert'
last_name = 'einstein'
full_name = f"{first_name} {last_name}"
print(full_name)
albert einstein
