Django

Setup

$ python -m venv ll_env
$ source ll_env/bin/activate
(ll_env)$ pip install Django
$ django-admin startproject ll_project .
$ python manage.py migrate
$ python manage.py runserver
Starting development server at http://127.0.0.1:8000/

Create App & Model

$ python manage.py startapp learning_logs
# learning_logs/models.py
class Topic(models.Model):
    text = models.CharField(max_length=200)
    date_added = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.text

Migrations & Admin

# settings.py: add 'learning_logs' to INSTALLED_APPS

$ python manage.py makemigrations learning_logs
$ python manage.py migrate
$ python manage.py createsuperuser

Template Inheritance

<!-- base.html (parent) -->
<p><a href="{% url 'learning_logs:index' %}">Learning Log</a></p>
{% block content %}{% endblock content %}
<!-- index.html (child) -->
{% extends 'learning_logs/base.html' %}
{% block content %}
<p>Learning Log helps you keep track...</p>
{% endblock content %}
Child template inherits nav from base, fills content block
