Document Structure

<!DOCTYPE>

Declares the HTML version. Must be the first line.

<!DOCTYPE html>

<html>

root element

The root element. Attribute lang sets language.

lang="en"

<head>

Container for metadata: <title> <meta> <link> <style> <base>

<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
  <link rel="stylesheet" href="styles.css">
</head>

<title>

Sets the browser tab title / SEO title.

My Page check browser tab

<meta>

charset / name / content

Provides metadata. Common attributes: charset, name, content, http-equiv.

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width">
<meta name="description" content="...">

<link>

Links external resources. Attributes: rel href type media

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico">
<link rel="preload" href="font.woff2" as="font">

<style>

Embeds CSS. Attribute media for responsive scoping.

<style>
  body { color: #fff; }
</style>

<base>

Sets base URL for all relative links. Attributes: href target

<base href="https://example.com/" target="_blank">

<body>

Contains all visible page content.

<body>
  <!-- visible content -->
</body>
