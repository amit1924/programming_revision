Forced Color Adjust

forced-color-adjust-*
forced-color-adjust-auto

Default. The browser may adjust colors when forced color mode is active (e.g., Windows High Contrast).

forced-color-adjust-none

Opts the element out of forced color mode adjustments, preserving the authored colors.

Use forced-color-adjust-none on elements where preserving custom colors is critical (buttons, badges, etc).

Prefers Contrast

contrast-more / contrast-less
contrast-more

Styles applied when the user has requested higher contrast in their OS settings. Use for stronger borders, darker text, etc.

This box has stronger borders in high contrast mode

contrast-less

Styles applied when the user has requested less contrast. Use for softer colors and reduced visual intensity.

This box adapts for reduced contrast preference

These utilities work automatically based on OS/browser contrast settings. No JavaScript needed for production.
